# 🏫 Classroom Availability and Update System

A web-based application that allows students to check classroom availability in real-time and enables Class Representatives (CRs) to update classroom usage dynamically.

---

## 📌 Project Overview

The Classroom Availability and Update System helps students easily identify free and occupied classrooms in real-time. It allows Class Representatives (CRs) to update classroom status dynamically, improving classroom utilization and reducing confusion.

---

## 🎯 Features

- Real-time classroom availability  
- Card-based classroom display  
- Status indicators (Free / Occupied)  
- CR dashboard for updating classroom status  
- Displays subject, start time, and end time  
- Simple and user-friendly interface  

---

## 🛠️ Technologies Used

- HTML, CSS, JavaScript  
- PHP  
- MySQL  
- XAMPP  

---

## ⚙️ Installation

1. Clone the repository  
2. Move project to `htdocs` (XAMPP)  
3. Start Apache and MySQL  
4. Create database `classroom_db`  
5. Create required tables  
6. Run project in browser  

---

## 🚀 Run

http://localhost/classroom-availability-and-update-system/

---

## 👨‍💻 Usage

### Student
- View classroom availability  
- Check Free / Occupied status  

### CR
- Select classroom  
- Enter subject and timing  
- Update classroom status  

---

## 📊 Workflow

1. Student views classrooms  
2. CR updates classroom  
3. Database updates  
4. UI reflects changes
